密码生成与管理工具

项目类型 安全工具
开发环境 Python 3.7+
适用场景 个人密码管理、高强度密码生成

该工具基于 Python 开发，集密码生成与管理于一体，采用本地加密存储方式，无需联网即可使用，保障密码信息安全。工具支持生成高强度随机密码，同时可对个人各类账号密码进行分类管理、模糊搜索和快速复制，解决密码记忆困难、安全性低等问题。

核心功能

  • 高强度密码生成：支持自定义密码长度（8-64 位）、字符类型（大小写字母、数字、特殊字符），生成不可预测的强密码
  • 本地加密存储：所有密码采用 AES 加密算法存储在本地 JSON 文件中，仅通过主密码解密，防止信息泄露
  • 密码分类管理：支持按平台（网站、APP、系统）分类存储密码，便于整理和查找
  • 模糊搜索功能：支持按平台名称、账号关键词搜索，快速定位目标密码
  • 一键复制：支持密码快速复制到剪贴板，方便粘贴使用
  • 主密码保护：设置唯一主密码，每次打开工具需验证主密码，防止他人非法访问
  • 密码强度检测：对用户手动输入的密码进行强度评分，提示安全风险

技术亮点

采用 cryptography 库实现 AES 加密算法，确保密码存储安全；密码生成使用 Python random 模块的加密级随机数生成器，保证密码随机性；数据存储采用 JSON 格式，结构清晰，便于备份和迁移；主密码验证采用哈希加盐存储，防止主密码泄露；支持剪贴板操作，通过 pyperclip 库实现密码一键复制；界面采用命令行交互，简洁直观，无需图形界面依赖；添加数据备份和恢复功能，防止密码数据丢失。
