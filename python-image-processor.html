图片批量处理工具

项目类型 图像处理工具
开发环境 Python 3.7+
适用场景 图片优化、批量处理、格式转换

该工具基于 Python Pillow 库开发，支持图片批量处理，无需专业图像处理软件即可完成格式转换、尺寸压缩、水印添加等操作。工具支持自定义处理参数，处理速度快，输出质量可控，适用于自媒体运营、电商图片优化、办公文档图片处理等多种场景，大幅提升图片处理效率。

核心功能

  • 批量格式转换：支持 JPG、PNG、GIF、BMP 等多种格式相互转换，可批量处理文件夹内所有图片
  • 尺寸压缩：支持按比例压缩或固定尺寸压缩，自定义压缩质量（1-100），平衡图片质量和文件大小
  • 水印添加：支持文字水印和图片水印，可自定义水印位置、透明度、大小、颜色
  • 图片特效：支持灰度处理、黑白反转、亮度调整、对比度调整等基础特效
  • 批量重命名：支持按自定义规则批量重命名处理后的图片，便于管理
  • 文件夹输出：处理后的图片自动保存到指定文件夹，保持原文件结构不变
  • 进度显示：实时显示处理进度、成功条数、失败条数，便于监控任务状态

技术亮点

基于 Pillow 库实现高效图像处理，支持多种图片格式的读写和编辑；采用多线程处理机制，大幅提升批量处理速度；支持递归遍历文件夹，处理子目录下的图片文件；参数配置通过 JSON 文件实现，灵活调整处理规则；添加图片质量无损压缩算法，在减小文件大小的同时保持清晰度；支持批量预览处理效果，避免误操作；异常处理机制完善，支持跳过损坏图片，确保任务正常完成。
